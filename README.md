# 🏥 Healthcare Revenue Dashboard

  A responsive web-based healthcare analytics dashboard built using HTML, CSS, JavaScript, and Chart.js to visualize hospital performance across branches, departments, services, and time periods.

This project simulates a real-world Business Intelligence dashboard similar to Power BI/Tableau — but built fully on the web.

# 🚀 Features

  📊 Interactive charts (Line, Bar, Pie, Doughnut)

  📈 KPI cards for Revenue, Patients, and Growth

  🗓 Multi-level filtering (Branch, Year, Month, Department, Service)

  📅 Day-wise patient visit analysis

  🌙 Dark mode toggle with animation

  📄 Export dashboard to PDF

  📱 Responsive full-screen layout

  🎨 Modern UI with smooth transitions

# 🧱 Project Architecture
  📁 project-folder
   ├── index.html      → Dashboard layout & components
   ├── style.css       → UI styling & responsive layout
   ├── script.js       → Data logic, filters & chart rendering
   ├── data.js         → Sample dataset (revenue & patient info)
 
# ⚙️ Tech Stack

  HTML5

  CSS3 (Flexbox + Grid layout)

  Vanilla JavaScript

  Chart.js for data visualization

  html2canvas + jsPDF for export functionality
